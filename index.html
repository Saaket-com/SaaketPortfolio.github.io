Hardware Projects

Electronics and embedded systems

Electronics / Multisim NI

12 V Power Supply

Built a 12 volt DC power supply using a transformer, bridge rectifier, filter, and LM317H voltage regulator IC. This project helped me understand how AC voltage is converted into stable DC voltage through practical design and simulation.

Analog Electronics / KiCad

Audio Level Visualizer

Built an audio level visualizer using the LM3914N-1 analog LED driver IC, an analog microphone, resistors, and 10 LEDs to display audio-based voltage levels in real time.

Arduino / Embedded

360 Degree Radar System

Created an Arduino-based radar system using ultrasonic sensors for object detection and a 360 degree radar-style visual interface built in Processing using Java.
